# Causal-Deep-Learning-ASD-Therapy
Valutazione causale della Robot-Enhanced Therapy (RET) per l'autismo tramite Deep Learning. Il progetto usa DragonNet e BCAUSS sul dataset DREAM per stimare l'effetto controfattuale dell'intervento robotico, isolando l'impatto reale sull'attenzione dei pazienti da bias e variabili confondenti.
